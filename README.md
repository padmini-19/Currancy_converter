# Currency Converter

A simple and responsive Currency Converter web application built using HTML, CSS, and JavaScript.

## Features

* Convert currencies instantly.
* Dynamic currency selection using dropdown menus.
* Country flags update automatically when a currency is selected.
* Fetches real-time exchange rates using a currency API.
* Responsive and user-friendly interface.

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Currency Exchange API

## Project Structure

```text
Currency-Converter/
│
├── index.html
├── style.css
├── app.js
├── codes.js
```

## How to Run

1. Download or clone the repository.
2. Open the project folder in VS Code.
3. Install the Live Server extension (optional).
4. Open `index.html` in your browser or run using Live Server.

## How It Works

1. Enter an amount.
2. Select the source currency.
3. Select the target currency.
4. Click the **Convert** button.
5. The application fetches the latest exchange rate and displays the converted amount.

## Future Improvements

* Add currency swap functionality.
* Display exchange rate history.
* Add more currencies and country flags.
